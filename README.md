# Zabbix Desktop Monitor 🖥📊
Десктопное приложение на Python (PyQt6) для удобного мониторинга, анализа и визуализации данных из системы Zabbix в режиме реального времени. Разработано в качестве альтернативы стандартному веб-интерфейсу для оперативного контроля сетевой инфраструктуры.
---
## 🚀 Основной функционал
### Обязательные требования (выполнено):
* **🔐 Безопасная авторизация:** Экран логина с проверкой учетных данных через официальный Zabbix API.
* **📋 Мониторинг узлов сети:** Интерактивный список всех хостов (`hosts`) с отображением текущих IP-адресов и статуса доступности.
* **📈 Визуализация метрик:** Встроенный графический виджет на базе `Matplotlib`, отображающий динамику загрузки CPU за последний час.
* **🔍 Умный фильтр:** Быстрый поиск устройств по имени, IP-адресу или технологической группе узлов.
* **📝 Логирование сбоев:** Автоматическая запись критических ошибок подключения и таймаутов API в файл `app_errors.log`.
### Бонусные возможности (выполнено):
* **🟢 Индикация статусов:** Динамическая цветовая маркировка (Зеленый — узел активен в мониторинге, Красный — деактивирован/недоступен).
* **⏱ Автообновление:** Автоматическая синхронизация данных с сервером каждые 30 секунд без необходимости перезапуска окон.
* **📥 Экспорт отчетов:** Возможность выгрузки текущей таблицы узлов сети со всеми статусами в формат `.csv` для отчетности.

## 🛠 Архитектура проекта
Проект реализован в соответствии с паттерном **MVC (Model-View-Controller)**, что обеспечивает четкое разделение логики и интерфейса:
* `app.py` — Главный файл приложения. Содержит интерфейс окон PyQt6 (Вход, Главная панель), контроллер логики и таймеры автообновления.
* `zabbix_backend.py` — Модуль интеграции с API. Отвечает за сетевые сессии, парсинг JSON-ответов от сервера и обработку ошибок.
* `app_errors.log` — Файл истории ошибок бэкенда (создается автоматически).

## 📦 Инструкция по развертыванию
Для запуска приложения локально на компьютере выполните следующие шаги:
### 1. Клонирование репозитория
```bash
git clone <https://github.com/yokous/DesktopZabbix.git>
cd zabbix_desktop
```
### 2. Создание и активация виртуального окружения (venv)
**Для Linux / macOS:**
```bash
python3 -m venv venv
source venv/bin/activate
```
**Для Windows (PowerShell):**
```powershell
python -m venv venv
.\venv\Scripts\Activate.ps1
```
### 3. Установка необходимых зависимостей
```bash
pip install PyQt6 matplotlib pyzabbix
```
### 4. Запуск приложения
```bash
python app.py
```
